# AnimalLingo Web App

React + TypeScript + Tailwind CSS web interface for AnimalLingo.

## Features

- 🎙️ **Record & Translate** - Capture pet sounds and get instant translations
- 💬 **Speak to Pets** - Convert human speech to pet-appropriate sounds  
- 🐕 **Pet Management** - Add and manage multiple pet profiles
- 📊 **Translation History** - Review past translations with confidence scores
- 📱 **Responsive Design** - Works on desktop, tablet, and mobile browsers

## Quick Start

```bash
cd web
npm install
npm run dev
```

Open http://localhost:5173

## Build for Production

```bash
npm run build
```

## API Integration

The web app connects to the backend API running at `http://187.77.6.208:3000`

Make sure the backend is running before starting the web app.
